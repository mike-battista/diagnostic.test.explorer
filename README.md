There is a test with scores ranging from 0 to 200. This tool allows you to explore how that test can be used to diagnose people with a disease. Play with the sliders to visualize:

* How people with and without the disease score on the test. Severity is how differently these populations score on average.

* How setting a diagnostic cutoff on the test changes its accuracy.

* How prevalence of the disease changes confidence in the results.

Click around for some hidden features. Created by Mike Battista using Claude. www.mikebattista.com
